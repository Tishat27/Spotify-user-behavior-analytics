# Spotify-user-behavior-analytics
# 🎵 Spotify User Behavior & Demographics Dashboard

## 📌 Project Overview
This project presents an interactive **Power BI Dashboard** analyzing user behavior, demographics, and subscription trends for Spotify. Built using official Spotify branding guidelines, this report turns raw survey data into actionable business insights.

## 🎯 Key Questions & Insights (Page 1: Demographics)

This first page addresses fundamental user demographic questions:

1. **User Demographics:** What is the distribution of users by age group and gender?
   * *Visual:* Pyramidal / Stacked Bar Chart (`User Demographics by Age Group & Gender`)
2. **Listening Devices:** What are the most popular devices used to access Spotify?
   * *Visual:* Bar Chart (`Most Used Listening Devices`)
3. **Content Preference:** How do preferences for Music vs. Podcasts vary across age groups and genders?
   * *Visual:* 100% Stacked Bar Chart with Small Multiples (`Preferred Content Type by Age Group`)
4. **Subscription Model:** What proportion of users are on Free vs. Premium plans across age groups?
   * *Visual:* 100% Stacked Bar Chart (`Subscription Plan Distribution by Age Group`)
